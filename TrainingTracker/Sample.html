      [ ID -129  ] - RAC/GI Support on Windows (Tools  and advance troubleshooting )
    
			   
		      
    
           		
		      
     Presenter  : STEPHAN.LANGE@ORACLE.COM, SAURABH.BAIJAL@ORACLE.COM, MITCHELL.FLATLAND@ORACLE.COM
          
 
			   
    
			   
             
    
				  
       Type:  Technical 
      PLA Family :  DB Systems
     
				  
      PLA Line : Database
      PLA Area : Scalability
    
			   
    
			   


               
    
                 

    
			     Description 
	            
     Each Session is on how to support RAC/GI on Windows.

I am intentionally keeping them short so that you aren't overwhelmed by too much information at once and 
also to give you the opportunity to pick and choose any topic(s) without having to go through them all.

The first session will provide information about the architecture of and how Oracle has implemented GI and RAC on Windows.
It will cover things such as "Windows services, registry, processes and threads, disk management , and more ". 
Understanding these Windows fundamentals and how they relate to the Oracle software on Windows is the basis knowledge 
needed to be able to understand and troubleshoot the kinds of issues that support has to deal with when  RAC/GI is on Windows.

The second  series is of talks and demos regarding supporting Oracle on Windows I will be giving a live demo of how to configure and install Windows for an RAC/ASM.

I will cover things like how to configure storage  for ASM ( and clusterware ), verify networking and user permissions  and then step through the GI and DB install.

The next topic in this series will be given by Mitch Flatland,  and it will be on "Windows OOB Processing"

In next session , I'll be back with a talk ( and demo ) about the tools available for supporting Oracle on Windows.

followed by Windows Tools to Troubleshoot Oracle & Networking on Windows,Upgrading/Patching on Windows,
"Using advanced tools from Sysinternals" - Saurabh Baijal,
"Advanced debugging on Windows using ADPLS/Windbg" - Mitch Flatland

- Stephan

Supporting RAC/GI on Windows - List of Sessions
 
1) Introduction to RAC/GI on Windows  								 (May 27, 2015 10:30 am     |  Eastern Daylight Time (New York, GMT-04:00)  |  1 hr s)
2) Installing RAC on Windows  		  								 (July 8th, 2015 11:00 am   |  Eastern Daylight Time (New York, GMT-04:00)  |  1 hr 30 mins )
3) Windows Tools needed/available for Troubleshooting RAC on Windows 
        &
 4) Troubleshooting Networking issues on RAC on Windows 			 (July 29th , 2015 11:00 am |  Eastern Daylight Time (New York, GMT-04:00)  |  1 hr 30 mins)
5) Troubleshooting Installation/Patching issues on RAC on Windows    (August 26, 2015 11:00 am  |  Eastern Daylight Time (New York, GMT-04:00)  |  1 hr 45 mins)
6) Using advanced tools from Sysinternals                            (September 2, 2015 11:00 am  |  Eastern Daylight Time (New York, GMT-04:00) | 1 hr 30 mins)
7) Advanced debugging on Windows using ADPLS/Windbg                  (September 16, 2015 11:00 am  |  Eastern Daylight Time (New York, GMT-04:00)| 1 hr 30 muns)
